# Stara Florencia (Restaurant Website)

A multi-page Bulgarian restaurant site with responsive navigation and modern styling.

**Live Demo:** https://theoldflorence.netlify.app/

## Features
- Responsive header with hamburger menu
- Pages: Home, Menu, About, Gallery, Contacts
- Warm Mediterranean-inspired palette and typography
- Lightweight: pure HTML, CSS, vanilla JS

## Tech Stack
- HTML5
- CSS3
- JavaScript (vanilla)

## Structure
- `index.html` — Home
- `menu.html` — Menu
- `about.html` — About
- `gallery.html` — Gallery
- `contacts.html` — Contacts
- `styles.css` — Global styles
- `img/` — Images

## Live Preview (VS Code)
1) Install the **Live Server** extension.
2) Open this folder in VS Code.
3) Right-click `index.html` → **Open with Live Server**.
4) The site will open at `http://127.0.0.1:5500/` (or similar).

## Quick View without Extensions
- Just double-click `index.html` to open in your browser.

## Deployment (GitHub Pages)
1) Push the repo to GitHub.
2) In repo **Settings → Pages**, select branch `main`, folder `/ (root)`.
3) Save; your site will be at `https://<your-username>.github.io/Resturant/`.

## Development
- Edit HTML/CSS directly; no build step.
- For mobile testing, use browser devtools → device toolbar.
